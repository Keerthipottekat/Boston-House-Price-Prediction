# 🏠 Boston House Price Prediction

This project uses **Machine Learning** to predict the median price of houses in Boston based on different housing features such as crime rate, number of rooms, tax rate, and distance to employment centers.

The project demonstrates the **complete machine learning pipeline including data preprocessing, model training, and prediction**.

---

## 📌 Project Overview

The Boston Housing dataset contains information collected by the U.S. Census Service concerning housing in Boston, Massachusetts.  
The dataset includes **506 records and 13 features** that describe different characteristics of houses and their environment.

The goal of this project is to **predict the median value of owner-occupied homes** using regression models.

---

## 📊 Dataset Features

Some important features used in the dataset include:

| Feature | Description |
|-------|-------------|
| CRIM | Per capita crime rate by town |
| ZN | Proportion of residential land zoned for lots |
| INDUS | Proportion of non-retail business acres |
| NOX | Nitric oxide concentration |
| RM | Average number of rooms per dwelling |
| AGE | Proportion of houses built before 1940 |
| DIS | Distance to employment centers |
| TAX | Property tax rate |
| PTRATIO | Pupil-teacher ratio |
| LSTAT | Percentage of lower status population |
